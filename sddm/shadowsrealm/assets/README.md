# ShadowsRealm SDDM assets

Drop a `wallpaper.png` (or `.jpg`) here to use it as the blurred login background.
The path is referenced by `background=` in `theme.conf`.

If no image is present, the greeter falls back to the solid Material You
background color from `Colors.qml`, so the theme still looks correct.

Tip: `install.fish` can copy your Caelestia wallpaper here automatically.
